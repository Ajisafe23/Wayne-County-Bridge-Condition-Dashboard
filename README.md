An ArcGIS Online dashboard visualizing bridge condition data across Wayne County for infrastructure planning and maintenance.
# Wayne County Bridge Condition Dashboard

[View Dashboard on ArcGIS Online](https://wayne.maps.arcgis.com/apps/dashboards/2084b584c8b642bc919535f39edd22e8)

## Overview
This dashboard was developed using **ArcGIS Online (AGOL)** to visualize and monitor **bridge conditions across Wayne County**. It provides an interactive and data-driven view of infrastructure health, helping engineers and planners identify maintenance needs, prioritize inspections, and allocate resources efficiently.

## Features
- Real-time visualization of bridge condition ratings and locations  
- Integration with county GIS databases  
- Interactive filters for bridge type, status, Deck Condition, etc.  
- Intuitive map-based navigation for quick data access  

## Tools and Technologies
- **ArcGIS Online (AGOL)**  
- **ArcGIS Dashboards**  
- **Geospatial Database (RIMS)**  
- **Data Visualization and Spatial Analysis**

## My Role
As GIS Manager and Application Developer, I designed and configured the dashboard, integrated bridge data from multiple sources, and ensured data accuracy and accessibility for county departments.

## Impact
The dashboard improves infrastructure management and transparency by allowing quick access to bridge condition data and visual insights for maintenance planning.
